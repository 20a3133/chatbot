コマンド一覧

キー操作

キー操作 操作内容
alt+/ コマンド入力欄にフォーカスする
alt+c テキストモードとキャンバスモードを切り替える
Ctrl+Enter 入力したコマンドを実行する
矢印キー フォーカスしているテキストボックスを任意方向に移動する
Shift+Backspace フォーカスしているテキストボックスを削除する

テキストモード

コマンド 操作内容
crtxt 座標(0,0)にフォントサイズ,フォントカラーがデフォルトのテキストボックスを作成する
crtxt/ x y/ fs size/ fc color/ 座標(x,y)にフォントサイズsize px,フォントカラーcolorのテキストボックスを作成する
x,y,sizeに数字,colorに色名(英語名)を入力する
/間に指定がない場合は何も記述しなくていい
fs size デフォルトのフォントサイズ,フォントカラーをsize pxにする
fc color デフォルトのフォントカラーをcolorにする
defalt fc/fs デフォルトのフォントサイズ,フォントカラーをブラウザのデフォルト設定にする

キャンバスモード

キー操作 操作内容
rect 描画する図形を矩形にする
line 描画する図形を直線にする
circle x 描画する図形を半径x pxの円にする
px x 描画の線の太さをx pxにする
color name 描画の線の色をname色にする
nameには色名(英語名)を入力する
erace 矩形の内側の描画を削除する
clear 全ての描画を削除する
